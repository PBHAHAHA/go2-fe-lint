## 使用第三方包说明： 
- ora: 终端 loading https://www.npmjs.com/package/ora
- cross-spawn：child_process 子进程下有一个spawn函数 可以调用系统的命令。在 linux、macos 上有效，但是windows上不生效，这个包就是解决系统的兼容问题
- fs-extra： 替代 Node.js 内置 fs 模块，更安全、更强大的文件操作库
