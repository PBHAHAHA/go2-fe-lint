# commitlint-config-go2

:::tip
 git 规范
:::

## 安装

```bash
npm install markdownlint-config-encode markdownlint --save-dev

```

## 使用

在 `commitlint.config.js` 中继承本包:

```js
module.exports = {
  extends: ['commitlint-config-go2'],
};
```
